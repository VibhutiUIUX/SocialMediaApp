# Social Media App

## Project Description

This is a simple Social Media Platform developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. The application allows users to register, log in, create posts, like posts, and add comments. It demonstrates the basic functionality of a social networking platform using full-stack web development.

---

## Features

- User Registration
- User Login
- Create New Posts
- View All Posts
- Like Posts
- Add Comments
- MongoDB Database Integration
- Responsive User Interface

## Installation

### Clone the repository

```bash
git clone https://github.com/VibhutiUIUX/SocialMediaApp.git
```

### Go to the project folder

```bash
cd SocialMediaApp
```

### Install dependencies

```bash
npm install
```

### Start MongoDB

Make sure MongoDB is running on:

```
mongodb://127.0.0.1:27017/socialmedia
```

### Start the server

```bash
node server.js
```

### Open the application

```
http://localhost:5000
```
